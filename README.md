# Tab Launcher

Chrome extension: search open tabs, bookmarks, and saved shortcuts from a quick overlay.


**Shortcuts:** In Settings (gear icon), add URL shortcuts with a short **key** (and optional label). In the launcher, type **`@`** plus that key—e.g. `@mail`—to list or jump to those shortcuts only. `@` by itself lists every shortcut.

**Auto open tab:** If there’s only one result, that tab or URL opens automatically.

**Open / close:** `Alt+Space` (change under `chrome://extensions/shortcuts`).

**Load locally:** Chrome → Extensions → Developer mode → Load unpacked → this folder.

**Search:** Type to filter tabs and bookmarks by title or URL. Your saved shortcuts also show up when their label or URL matches.



Does not run on `chrome://`, the Web Store, or other restricted pages—use a normal website tab.
